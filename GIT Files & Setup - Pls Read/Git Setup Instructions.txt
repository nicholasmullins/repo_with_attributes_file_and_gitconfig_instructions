 After you clone the repo:

1. Go to your repo folder and hit CMD-SHIFT-PERIOD. This will show your hidden files
2. Open your .git directory
3. In the .git directory, open the 'config' file and copy this code and paste it at the bottom:

[merge "ours"]
        driver = true

4. In the .git directory, replace your 'info' directory with the 'info' directory in this folder.
5. Close your .git directory
