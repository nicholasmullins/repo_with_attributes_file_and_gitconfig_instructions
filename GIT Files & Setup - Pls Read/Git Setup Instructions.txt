 After you clone the repo:

1. Go to your repo folder and hit CMD-SHIFT-PERIOD. This will show your hidden files
2. Open your .git directory
3. In the .git directory, open the 'config' file and copy this code and paste it at the bottom:

[merge "ours"]
        driver = true


4. Close 'config' file

5. Copy the 'info' directory in this Git Setup folder. Paste it into your .git directory, replacing your current 'info' directory.

6. Your new 'info' directory should now contain two files: exclude and attributes
